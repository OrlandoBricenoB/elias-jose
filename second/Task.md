## Siguientes pasos

Ahora que ya tienen conocimientos para crear un repositorio de Github, van a crear el suyo propio y resolver lo siguiente:

- Deben construir un sitio web con un HTML vinculado a su JavaScript y tendrán que construir las siguientes funciones:

* Una función que reciba un número y regrese true si es par, su nombre puede ser: isEven()
  * Si lo que recibió por parámetro no es un número, debe regresar un Error('El parámetro debe ser un número')

* Una función que exclame una frase, recibe un texto por parámetro y la encierra entre ¡ y !
  Ejemplo:
    exclaim('Saludos') // ¡Saludos!

* Una función que limpie los espacios en blanco en un texto al inicio y al final:
  Ejemplo:
    cleanSpaces('   ¿Qué pasa, brother? ¿Mucho espacio?    ') // '¿Qué pasa... espacio?'

Esto deben entregarlo antes del viernes 13.
